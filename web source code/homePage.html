Welcome to the Hogwarts Database

CS 340 Final Project

A database driven website using HTML, CSS, PHP and MySQL


This was my final project for CS 340 (Introduction to Databases) at OSU, which required a database with at least 4 entities and 4 relationships, one of which was to be a many-to-many relationship. All source code is available on my GitHup repository here.



Project Contents:

The Basics: Adding data to entity/relationship tables:

  • Student Blood Status
  • Hogwart's Houses
  • Hogwart's Subjects
  • Hogwart's Teachers

Please note, adding students to the students_tbl and adding students to subjects (takes_subject) takes place in the sections below.
